# Tafsir-Juza-Amma
Kebanyakan orang hafal ayat-ayat Juz 'Amma sejak kecil — tapi berapa banyak yang benar-benar menyelami maknanya?

Proyek ini lahir dari satu pertanyaan sederhana:

Bagaimana jadinya jika kita membaca tafsir Al-Qur'an bukan sebagai kewajiban akademik, tapi sebagai percakapan langsung dengan semesta?

Hasilnya adalah 38 tafsir tematik — dari Al-Fatihah hingga An-Nas — yang ditulis dengan pendekatan kontemporer: menghubungkan setiap ayat dengan realita kehidupan sehari-hari, psikologi modern, dan pertanyaan-pertanyaan besar yang sebenarnya kita semua pikirkan
